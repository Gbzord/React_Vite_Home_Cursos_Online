## 🚀 Como Começar

### 1. Instalar dependências
```bash
cd meu-app
npm install
```

### 2. Iniciar dev server (Vite)
```bash
npm run dev
```

Abrirá em: **http://localhost:3000** com HMR ativo!

### 3. Build para produção
```bash
npm run build
```

Gera otimizado em `/dist`
